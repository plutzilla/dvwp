== Installation ==

1. Upload the plugin files to `/wp-content/plugins/woocommerce-social-share/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings > Social Share to configure the plugin
4. Enter your Pro license key to unlock premium features
